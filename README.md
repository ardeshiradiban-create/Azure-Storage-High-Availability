# Implementing-Azure-File-Sync-and-Backup

## Introduction
As an **IT & Project Coordinator** at **Greenwall Companies**, I designed this project to demonstrate the deployment of secure, resilient, and scalable data storage solutions in the cloud. Leveraging over 15 years of experience in IT infrastructure, I implemented a solution that ensures **Data Durability** and provides centralized **Cloud File Services**, replacing traditional on-premises file servers with modern Azure architecture.
## Technical Tasks
- **Storage Account Deployment**: Configured a storage account with **GRS (Geo-redundant storage)** for maximum availability.
- **Azure File Shares**: Created a centralized file share to replace traditional on-premises file servers.
- **Access Tiers**: Leveraged Azure Access Tiers (Hot/Cool) to balance high-performance needs with budget efficiency.
